# مدير سجل الحافظة

إضافة لإدارة سجل الحافظة في NVDA مستوحاة من برنامج Ditto.

تتتبع كل ما تنسخه — نصوص وملفات وروابط وعناوين بريد إلكتروني — مع مراقبة فورية للحافظة تعتمد على الأحداث.

## الميزات

* **مراقبة فورية للحافظة** باستخدام مستمع حافظة Windows (مبني على الأحداث، بدون استقصاء دوري).
* **دعم الملفات**: الملفات المنسوخة من مستكشف Windows تظهر في السجل ويمكن لصقها كملفات فعلية (تنسيق CF_HDROP).
* **تصنيف تلقائي**: يتم تصنيف العناصر تلقائياً كملفات ومجلدات أو روابط أو بريد إلكتروني.
* **بحث وتصفية**: بحث نصي كامل وتصفية حسب المجموعة في نافذة السجل.
* **تثبيت العناصر المهمة** حتى لا تُحذف عند الوصول للحد الأقصى.
* **تحديد متعدد ولصق**: حدد عناصر متعددة بـ Ctrl+مسافة والصقها بترتيب التحديد.
* **إلحاق بالحافظة**: إلحاق النص المحدد بمحتوى الحافظة الحالي.
* **إعادة ترتيب العناصر**: حرّك العناصر للأعلى/الأسفل بـ Shift+الأسهم، أو للأعلى/الأسفل عبر قائمة السياق.
* **حفظ العناصر**: احفظ أي عنصر كملف نصي (.txt) أو مستند Word (.docx) من قائمة السياق.
* **مؤشر اللصق**: العناصر التي تم لصقها تُعلَّم بـ "Pasted:" في القائمة.
* **اللصق المدمج**: عند لصق عناصر متعددة معاً، يُحفظ النص المدمج كعنصر جديد.
* **سجل دائم**: يُحفظ السجل بين جلسات NVDA (قابل للتخصيص).
* **مجموعات مخصصة**: إنشاء وإدارة مجموعات مخصصة لتنظيم العناصر.
* **محرر القصاصات**: أنشئ قصاصات جديدة من الصفر أو حرر العناصر الموجودة باستخدام محرر نصوص مدمج، مع تحذير من التغييرات غير المحفوظة.
* **تحديد الكل**: اضغط Ctrl+A في نافذة السجل لتحديد جميع العناصر دفعة واحدة.
* **قابل للتخصيص**: لوحة إعدادات في تفضيلات NVDA للحد الأقصى للعناصر والحفظ والإعلانات.

## اختصارات لوحة المفاتيح العامة

| الاختصار | الإجراء |
|---|---|
| NVDA+A | فتح نافذة سجل الحافظة |
| NVDA+C | الإعلان عن محتوى الحافظة الحالي |
| NVDA+Shift+A | إلحاق النص المحدد بآخر عنصر في الحافظة |
| NVDA+Alt+سهم لأعلى | الانتقال إلى العنصر السابق في السجل |
| NVDA+Alt+سهم لأسفل | الانتقال إلى العنصر التالي في السجل |
| NVDA+Alt+V | لصق العنصر المحدد حالياً |
| NVDA+Alt+Enter | نسخ العنصر المحدد حالياً إلى الحافظة |
| NVDA+Alt+Delete | حذف العنصر المحدد حالياً |
| NVDA+Alt+P | تثبيت/إلغاء تثبيت العنصر المحدد حالياً |
| NVDA+Alt+X | مسح كل سجل الحافظة |
| NVDA+Shift+C | فتح المحرر لكتابة قصاصة جديدة |
| NVDA+Alt+G | تعيين مجموعة للعنصر المحدد حالياً |

## اختصارات لوحة المفاتيح في النافذة

عند فتح نافذة سجل الحافظة:

| الاختصار | الإجراء |
|---|---|
| Enter | لصق العنصر/العناصر المحددة |
| Ctrl+مسافة | تبديل التحديد المتعدد (يراعي ترتيب التحديد) |
| Delete | حذف العنصر/العناصر المحددة |
| Ctrl+A | تحديد جميع العناصر |
| Ctrl+E | تحرير العنصر المحدد |
| Ctrl+P | تثبيت/إلغاء تثبيت العنصر |
| Ctrl+G | تعيين المجموعة |
| Shift+سهم لأعلى | تحريك العنصر لأعلى في القائمة |
| Shift+سهم لأسفل | تحريك العنصر لأسفل في القائمة |
| مفتاح التطبيقات | فتح قائمة السياق |
| Alt+S | التركيز على حقل البحث |
| Alt+U | التركيز على تصفية المجموعة |
| Alt+G | زر تعيين المجموعة |
| Escape | إغلاق النافذة |

## خيارات قائمة السياق

انقر بزر الماوس الأيمن أو اضغط مفتاح التطبيقات على أي عنصر:

* لصق
* نسخ إلى الحافظة
* تحرير
* **حفظ باسم** قائمة فرعية: ملف نصي (.txt)، مستند Word (.docx)
* **نقل إلى** قائمة فرعية: الأعلى، الأسفل
* تثبيت / إلغاء التثبيت
* تعيين المجموعة
* حذف

## الإعدادات

متاحة في قائمة NVDA > التفضيلات > الإعدادات > سجل الحافظة:

* **الحد الأقصى لعدد عناصر السجل**: حدد الحد (10–10,000، الافتراضي 500). عند تجاوزه، يُحذف أقدم عنصر غير مثبت.
* **حفظ السجل بين جلسات NVDA**: الاحتفاظ بالسجل عبر عمليات إعادة التشغيل (الافتراضي: مفعّل).
* **الإعلان عند نسخ نص جديد**: نطق معاينة عند اكتشاف محتوى جديد (الافتراضي: معطّل).
